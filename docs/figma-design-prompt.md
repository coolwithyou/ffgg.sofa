## 프로젝트 개요

**서비스명**: faq.guide
**도메인**: faq.guide
**한 줄 설명**: 소규모 사업자를 위한 가장 쉬운 FAQ 가이드 챗봇 서비스

### 핵심 철학

> "복잡한 기계가 아닌, 누구나 만질 수 있는 장난감"

- 타겟: 디지털에 익숙하지 않은 소규모 사업자
- 목표: 5분 안에 FAQ 챗봇 만들기
- 경쟁 차별점: 채널톡 같은 CS 챗봇이 아닌, "도슨트/가이드" 역할

## 디자인 레퍼런스

### 첨부 이미지 참조 (Readymag 스타일)

핵심 특징:
- **Flat**: 그림자 최소화, 평면적
- **Colorful**: 명확하고 과감한 포인트 컬러
- **Bold**: 두꺼운 폰트, 강한 대비
- **Minimal**: 요소 최소화, 여백 활용

### 피해야 할 것
- 복잡한 대시보드 느낌
- 그라데이션, 글래스모피즘
- 빽빽한 정보 배치
- 개발자/전문가용 느낌

## 디자인 원칙

### 1. 선택지는 최대 3개
한 화면에서 사용자가 선택할 수 있는 옵션은 3개 이하

### 2. 다음 할 일이 항상 보인다
RPG 퀘스트처럼 "지금 뭘 해야 하는지" 명확하게

### 3. 깊이는 1단계까지
모든 기능은 2클릭 이내 도달

### 4. 전문 용어 없음
"데이터셋", "청크", "임베딩" 같은 단어 절대 노출 안 함

### 5. 여백은 자신감
빈 공간은 "우리는 이것만 잘해"라는 메시지

## 요청 1: 랜딩 페이지 (faq.guide)

### 구조

```
[Header]
로고: faq.guide (점은 코랄 컬러)
메뉴: 없음 또는 최소 (가격, 로그인)

[Hero Section]
헤드라인: "질문에 답하느라 지치셨나요?"
서브: "파일 하나면 AI 가이드 완성. 5분이면 충분해요."
CTA: "무료로 시작하기" (코랄 버튼)
서브CTA: "데모 보기" (텍스트 링크)

[How it Works] - 3단계
1. 파일 올리기 → 2. AI가 정리 → 3. 공유하기
각 단계는 심플한 일러스트 또는 아이콘

[Social Proof]
"00개 사업장이 사용 중" 또는 고객 로고

[Pricing] - 2개 플랜만
Free: 핵심 기능
Pro: 확장 기능 + 가격

[Footer]
최소한의 링크
```

### 분위기
- Readymag 레퍼런스처럼 대담하고 컬러풀
- 스크롤 시 자연스러운 애니메이션
- 모바일 우선 반응형

## 요청 2: Studio 화면 (faq.guide/studio)

### 핵심 개념: 상태 기반 UI

화면이 "현재 상태"에 따라 자동으로 달라짐.
메뉴를 탐색하는 게 아니라, 시스템이 맥락에 맞는 화면을 보여줌.

### 상태 정의

| 상태 | 조건 | 화면 |
|------|------|------|
| **Empty** | 가이드 0개 | 시작 유도 |
| **Processing** | 생성 중 | 진행 상태 |
| **Ready** | 완성, 미공개 | 공개 유도 |
| **Live** | 공개됨 | 대시보드 |

### 메뉴 구조 (3개만)

```
[사이드바]
📚 가이드북    ← 가이드 목록, 추가, 편집
🎨 내 페이지   ← 디자인, 미리보기, 공개 설정
⚙️ 설정       ← 계정, 결제, 알림
```

---

### 상태별 화면 설계

#### Empty 상태 (첫 방문)

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  [사이드바]              [메인 영역]                 │
│                                                     │
│  📚 가이드북                    📄                  │
│  🎨 내 페이지                                       │
│  ⚙️ 설정              가이드를 만들어 시작하세요      │
│                                                     │
│                    ┌─────────────┐  ┌─────────────┐ │
│                    │ Q&A 직접 작성│  │ 파일로 만들기│ │
│                    └─────────────┘  └─────────────┘ │
│                                                     │
│                      샘플 가이드 둘러보기 →          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- 중앙에 큰 아이콘과 유도 문구
- 두 가지 시작 방법 제시
- 하단에 샘플 보기 링크

#### Processing 상태 (파일 업로드 후)

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                      ⏳                             │
│                                                     │
│              가이드를 만들고 있어요                  │
│              "옻칠 입문서.pdf"                       │
│                                                     │
│         ████████████░░░░░░░░░░░░  60%              │
│                                                     │
│           예상 시간: 약 2분 남음                     │
│                                                     │
│         완성되면 알림을 보내드릴게요                 │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- 진행률 바
- 파일명 표시
- 예상 시간
- 알림 옵션

#### Ready 상태 (완성, 미공개)

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                      ✅                             │
│                                                     │
│              가이드가 준비됐어요!                    │
│                                                     │
│       ┌─────────────┐    ┌─────────────┐           │
│       │  미리보기   │    │   공개하기   │ (코랄)    │
│       └─────────────┘    └─────────────┘           │
│                                                     │
│                 나중에 공개할게요 →                  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- 축하 메시지
- 미리보기 / 공개하기 두 옵션
- 공개하기가 Primary (코랄)

#### Live 상태 (공개됨, 메인 대시보드)

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   내 페이지                                         │
│   faq.guide/ottchil                    [복사] [열기]│
│                                                     │
│   ─────────────────────────────────────────────     │
│                                                     │
│   📊 오늘                                           │
│   ┌──────────┐ ┌──────────┐ ┌──────────┐           │
│   │ 방문 12명 │ │ 질문 8개  │ │ 만족 95% │           │
│   └──────────┘ └──────────┘ └──────────┘           │
│                                                     │
│   ─────────────────────────────────────────────     │
│                                                     │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│   │가이드 추가│ │페이지 꾸미기│ │  설정   │            │
│   └─────────┘  └─────────┘  └─────────┘            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- 상단: 공개 URL + 빠른 액션
- 중간: 간단한 통계 (3개만)
- 하단: 다음 할 일 카드들

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   내 페이지                                         │
│                                                     │
│   ┌─────────────────┐  ┌─────────────────────────┐ │
│   │                 │  │                         │ │
```

- 좌측: 실시간 미리보기

## 컴포넌트 가이드라인

### 버튼

| 타입 | 스타일 | 용도 |
|------|--------|------|
| Primary | 코랄 배경, 흰 텍스트 | 주요 CTA (1개만) |
| Secondary | 테두리, 검정 텍스트 | 보조 액션 |
| Ghost | 텍스트만 | 덜 중요한 액션 |

- border-radius: 12px
- 패딩: 12px 24px (기본)

### 카드

- border-radius: 16px
- 패딩: 24px
- 호버 시 미세한 상승 효과
- 그림자 최소화

### 입력 필드

- border-radius: 8px
- 라벨은 항상 위에
- 포커스 시 코랄 테두리
