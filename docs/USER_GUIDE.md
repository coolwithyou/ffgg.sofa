## 1. 시작하기

### 1.1 회원가입

1. [SOFA 홈페이지](https://sofa.ai)에서 "무료로 시작하기" 클릭
2. 이메일, 비밀번호 입력
3. 회사 정보 입력
4. 이용약관 동의 후 가입 완료

> **TIP**: 14일 무료 체험 기간 동안 모든 기능을 사용할 수 있습니다.

### 1.2 대시보드 접근

가입 후 자동으로 대시보드로 이동합니다. 대시보드에서는:
- 전체 현황 요약
- 최근 대화 목록
- 문서 처리 상태

를 확인할 수 있습니다.

## 2. 문서 업로드

### 2.1 지원 형식

| 형식 | 확장자 | 최대 크기 |
|------|--------|-----------|
| PDF | .pdf | 20MB |
| Word | .docx | 10MB |
| Excel | .xlsx, .csv | 10MB |
| 텍스트 | .txt | 5MB |

### 2.2 업로드 방법

1. 좌측 메뉴에서 "문서 관리" 클릭
2. "파일 업로드" 버튼 클릭 또는 드래그앤드롭
3. 업로드 완료 후 자동 처리 시작

### 2.3 처리 과정

```
업로드됨 → 처리 중 → 검토 대기 → 승인됨
```

1. **업로드됨**: 파일이 서버에 저장됨
2. **처리 중**: AI가 문서 분석 및 청킹 진행
3. **검토 대기**: 내부 검토 후 승인 예정
4. **승인됨**: 챗봇에서 사용 가능

### 2.4 문서 작성 팁

더 좋은 챗봇 응답을 위한 문서 작성 팁:

- **Q&A 형식**: "Q: 배송비는 얼마인가요? A: 3만원 이상 무료배송입니다."
- **명확한 제목**: 섹션별 명확한 제목 사용
- **구체적인 내용**: 모호한 표현 대신 구체적인 정보 기재
- **최신 정보**: 주기적으로 문서 업데이트

## 3. 챗봇 테스트

### 3.1 테스트 페이지

1. 좌측 메뉴에서 "챗봇 테스트" 클릭
2. 하단 입력창에 질문 입력
3. 응답 확인 및 품질 평가

### 3.2 응답 품질 개선

응답이 만족스럽지 않다면:

1. **문서 확인**: 해당 정보가 업로드된 문서에 있는지 확인
2. **문서 보완**: 누락된 정보 추가 후 재업로드
3. **문의**: 지속적인 문제 시 고객 지원 문의

## 4. 웹 위젯 설치

### 4.1 임베드 코드 복사

1. 좌측 메뉴에서 "설정" 클릭
2. "위젯 설정" 탭 선택
3. "임베드 코드 복사" 버튼 클릭

### 4.2 웹사이트에 설치

복사한 코드를 웹사이트 HTML의 `</body>` 바로 위에 붙여넣기:

```html
<!-- SOFA 챗봇 위젯 -->
<script src="https://sofa.ai/widget.js"></script>
<script>
  SofaWidget.init({
    tenantId: 'your-tenant-id'
  });
</script>
</body>
```

### 4.3 커스터마이징

```javascript
SofaWidget.init({
  tenantId: 'your-tenant-id',
  primaryColor: '#FF6B00',  // 메인 색상
  position: 'bottom-right', // 위치
  welcomeMessage: '안녕하세요! 무엇을 도와드릴까요?'
});
```

**옵션**:
- `primaryColor`: 위젯 메인 색상 (HEX 코드)
- `position`: `bottom-right` | `bottom-left`
- `welcomeMessage`: 첫 인사 메시지

## 5. 카카오톡 연동

### 5.1 사전 준비

1. 카카오 비즈니스 채널 필요
2. 카카오 오픈빌더 접근 권한

### 5.2 연동 단계

#### Step 1: SOFA 설정

1. SOFA 대시보드 → 설정 → 카카오 연동
2. "스킬 서버 URL" 복사

#### Step 2: 카카오 오픈빌더 설정

1. [카카오 오픈빌더](https://openbuilder.kakao.com) 접속
2. 챗봇 생성 또는 선택
3. 스킬 → 스킬 생성
4. URL에 복사한 스킬 서버 URL 붙여넣기
5. 메서드: POST
6. 저장

#### Step 3: 시나리오 연결

1. 시나리오 → 폴백 블록 선택
2. 스킬 데이터 사용 → 생성한 스킬 선택
3. 저장 및 배포

### 5.3 테스트

1. 카카오톡에서 채널 검색
2. 채팅 시작
3. 질문 입력 후 응답 확인

## 6. 설정 및 관리

### 6.1 기본 설정

- **회사명 변경**: 설정 → 기본 정보
- **담당자 정보**: 설정 → 기본 정보

### 6.2 보안 설정

- **비밀번호 변경**: 설정 → 보안
- **2단계 인증**: 설정 → 보안 → 2FA 활성화 (권장)

### 6.3 알림 설정

- **이메일 알림**: 일일 리포트, 에러 알림
- **설정 위치**: 설정 → 알림

## 자주 묻는 질문

### Q: 문서 업로드 후 얼마나 걸리나요?
A: 일반적으로 5-10분 내에 처리됩니다. 대용량 문서는 더 오래 걸릴 수 있습니다.

### Q: 챗봇 응답이 정확하지 않아요.
A: 업로드된 문서에 해당 정보가 명확히 포함되어 있는지 확인해주세요. Q&A 형식으로 작성하면 더 정확한 응답을 얻을 수 있습니다.

### Q: 위젯이 표시되지 않아요.
A: 브라우저 개발자 도구에서 콘솔 에러를 확인해주세요. CORS 에러가 발생하면 도메인 설정이 필요합니다.

### Q: 카카오톡 응답이 느려요.
A: 카카오톡은 5초 제한이 있어, 복잡한 질문은 시간이 걸릴 수 있습니다. 지속적인 지연 시 고객 지원에 문의해주세요.
