## 4. 2024-2025 트렌드

### 4.1 AI 기반 기능

- **AI 페이지 생성**: Beacons의 AI가 SNS 분석 후 자동 페이지 구성
- **AI 글쓰기 어시스턴트**: 블록 설명, 제목 자동 생성
- **스마트 링크 제안**: 추가할 링크 자동 추천
- **AI 디자인 추천**: 브랜드 컬러 기반 테마 제안

### 4.2 커머스 통합 강화

- **통합 결제**: Stripe, PayPal 직접 연동
- **디지털 상품 판매**: PDF, 영상 강의, 템플릿
- **구독/멤버십**: 반복 결제 모델
- **제휴 마케팅**: 어필리에이트 링크 관리

### 4.3 임베드 확장

- **노코드 통합**: Zapier, Make(Integromat) 연동
- **라이브 스트리밍**: Twitch, YouTube Live 임베드
- **실시간 위젯**: 방문자 수, 매출 현황
- **예약 시스템**: Calendly, Cal.com 등 통합

### 4.4 분석/인사이트

- **클릭 히트맵**: 어떤 블록이 주목받는지
- **전환 추적**: 클릭 → 구매 퍼널
- **A/B 테스트**: 블록 배치 실험
- **리텐션 분석**: 재방문율

## 6. 구현 권장사항

### 6.1 기술 스택

- **DnD 라이브러리**: `@dnd-kit/core` (접근성, 성능)
- **상태 관리**: Context API + Zustand
- **데이터 저장**: JSONB 필드 (`publicPageConfig.blocks`)
- **검증**: Zod 스키마

### 6.2 UX 패턴

- **클릭 + 드래그**: 두 가지 추가 방식 지원
- **실시간 미리보기**: 편집 즉시 반영
- **블록 설정 패널**: 선택 시 사이드바에 설정 표시
- **언두/리두**: Ctrl+Z, Ctrl+Y 지원 (추후)

### 6.3 데이터 구조 예시

```typescript
interface Block {
  id: string;           // nanoid
  type: BlockType;      // 'header' | 'chatbot' | 'link' | ...
  order: number;        // 순서
  visible: boolean;     // 표시 여부
  config: BlockConfig;  // 타입별 설정
}

interface LinkBlockConfig {
  url: string;
  title: string;
  description?: string;
  thumbnail?: string;
  style: 'default' | 'featured' | 'outline';
}
```
